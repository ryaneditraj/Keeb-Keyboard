# Keeb-Keyboard
I'm Building a keyboard for hackclub KEEB YSWS

Ive built a keyboard while following the KEEB documentation and other tutorials as well. It is a 60% keyboard type layout but ive added Arrow keys and then a pico on top. I really love typing and ive always wanted to build a mechanical keyboard.

Ive built various hardware projects before but ive never designed a pcd and schematic. I tought it would be fun to learn PCB Design, CAD and some basic schematics and wiring in KiCAD. 

I created schematic in ki cad connected them all up using the labels for each row and column.

later i brought it into PCB And aligned PCB as per a 60% keyboard and later on added arrows. I tought of mounting the raspberry pi on the back but i didnt really know how to do that or if that was possible so i just mounted it in the front.

I made the case in onshape, i am going to do some PE Foam and glue in the bottom so it is going to be glue mounted. To be honest i knew about the journalling thing only after i came to the submissions page, It was due to the fact that i was following it step by step i missed that part.
